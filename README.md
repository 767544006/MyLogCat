MyLogCat
======
MyLogCat 是在安卓手机上查看日志的库，这种实时查看日志的方式，可以快速定位测试人员或者用户反馈的错误问题。
    
## 特点：
  1. 支持Androd 2.3以上系统
  2. 无需root权限
  3. 不同级别的日志显示不同的颜色
  4. 可以过滤各种层级的log日志
  5. 根据正则去匹配tag的日志
  6. 保存，分享，清空当前输出日志
    
## 使用：
```
dependencies {
    compile 'com.gn.logcatlibrary:mylogcatlibrary:1.0.2'
}
```
